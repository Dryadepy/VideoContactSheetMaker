Default{X}.jpg shows the output of using default profiles. You can use them like this

VideoContactSheetMaker -i "C:\sample.mp4" -p default{X}

Such as:
VideoContactSheetMaker -i "C:\sample.mp4" -p default1
VideoContactSheetMaker -i "C:\sample.mp4" -p default2
VideoContactSheetMaker -i "C:\sample.mp4" -p default3
VideoContactSheetMaker -i "C:\sample.mp4" -p default...
